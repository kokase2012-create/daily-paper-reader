## 摘要
红外小目标检测（IRSTD）在复杂背景混杂中精确定位微弱目标面临固有挑战。尽管已有一定进展，现有方法通常遵循传统的下采样特征策略，丢失小目标的细节，导致性能不理想。本文提出了 Na-IRSTD，一种用于 IRSTD 的原始分辨率特征提取与融合框架。该框架巧妙地引入原始分辨率特征，以保留微弱目标线索，克服现有红外方法中的分辨率限制，并显著提升模型定位小目标的能力。我们还引入了一种有效的 token 减少与选择策略，该策略能够高精度、高置信度地选择目标 patch，在有效降低相对于密集处理的原始分辨率 patch token 数量的同时，增强特征的低层次细节，从而避免过高的计算负担。大量实验验证了我们的 token 减少与选择策略在多个公开数据集上的鲁棒性与有效性。最终，我们的 Na-IRSTD 模型在四个基准测试上实现了当前最先进的性能。

## Abstract
Infrared small target detection (IRSTD) faces the inherent challenge of precisely localizing dim targets amid complex background clutter. While progress has been made, existing methods usually follow conventional strategies to downsample features and discard small targets' details, resulting in suboptimal performance. In this paper, we present Na-IRSTD, a native-resolution feature extraction and fusion framework for IRSTD. This framework elegantly incorporates native-resolution features to preserve subtle target cues, overcoming the resolution limitations of existing infrared approaches and significantly improving the model's ability to localize small targets. We also introduce an effective token reduction and selection strategy, which selects target patches with high accuracy and confidence, boosting the low-level details of the feature while effectively reducing native-resolution patch tokens compared to dense processing, thereby avoiding imposing an unbearable computational burden. Extensive experiments demonstrate the robustness and effectiveness of our token reduction and selection strategy across multiple public datasets. Ultimately, our Na-IRSTD model achieves state-of-the-art performance on four benchmarks.

## 一、研究问题与背景

红外小目标检测（Infrared Small Target Detection, IRSTD）旨在从红外图像中检测面积极小、亮度微弱的目标。这类目标通常尺寸小于 32×32 像素，纹理稀疏、对比度极低，易被复杂背景（如云层、海浪、城市结构）淹没。  
现有主流检测方法多采用 **下采样（downsampling）** 的层级特征提取策略。然而，下采样虽降低计算量，却会丢失这些细小目标的分辨率信息，造成检测性能下降。  
论文基于此痛点提出核心问题：**能否在不显著增加计算负担的前提下，保留和融合原生分辨率特征，以增强对微弱小目标的定位精度？**

## 二、方法论：Na-IRSTD 框架

### 1. 核心思想
Na-IRSTD 提出一种 **原生分辨率特征提取与融合框架**，在常规模型下采样之前保留高分辨率信息，同时通过 **令牌（token）选择策略** 有效减少冗余计算。核心思想为：
- **保留原生分辨率细节**：在下采样前提取关键特征，避免目标信号坍缩。
- **高效令牌选择**：基于 patch 的相关性评分，仅选择最具目标特征的 patch 进行进一步处理。
- **多尺度融合**：将选出的高分辨率特征注入到下采样语义路径，实现分辨率与语义信息兼顾。

### 2. 结构组成
框架包含两条互补路径：
- **原生分辨率支路（Native-resolution branch）**
  - 以图像分块（patch）为单位进行卷积特征提取（Patchwise Detail Extraction, PDE）。
  - 通过自注意力机制的 **Global Patch Mixer (GPM)** 建模跨 patch 的全局关联。
  - 输出高分辨率特征令牌。
- **层级主干（Hierarchical backbone encoder）**
  - 类似 DETR 编码器，用于提取全图语义特征。

### 3. 关键技术环节

#### （1）Patchwise Detail Extraction (PDE)
- 将输入图像划分为不重叠的 P×P patch；
- 对每个 patch 独立进行卷积编码，保留局部红外细节。

#### （2）Global Patch Mixer (GPM)
- 对每个 patch 进行池化后转换为令牌 token；
- 利用自注意力实现空间长距离依赖建模；
- 保持复杂度从 **O((H×W)²)** 降为 **O((H/P×W/P)²)**。

#### （3）令牌选择策略（Top-K Token Reduction）
- 为每个 patch 预测相关性评分（由一个 MLP 输出）；
- 使用基于高斯函数的软标签（Soft Label）对 patch 进行监督；
- 从 N 个 patch 中选择前 K 个最高分 patch；
- 实现超过 90% 的令牌减少（例如 N=64，K=5 时减少 92.2%）。

#### （4）跨尺度注入（Cross-scale Injection）
- 将选出的 patch 与主干特征对应对齐；
- 用 Cross-Attention 融合语义特征与原生分辨率特征；
- 通过可学习系数 α 控制融合强度。

#### （5）两阶段训练策略（Two-stage Optimization）
1. **阶段一**：训练原生分辨率支路及评分 MLP，仅预测 patch 相关性。  
2. **阶段二**：冻结评分模块，训练完整检测网络进行分割任务（使用 BCE + Dice Loss）。
