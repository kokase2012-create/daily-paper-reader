## 摘要
可靠的无人机（UAV）目标检测需要在光照变化、运动模糊和场景动态导致RGB线索减弱的情况下保持鲁棒性。热成像长波红外（LWIR）传感器在弱光环境下能够保持对比度，而事件相机能以微秒级分辨率捕捉时间边缘，但目前尚无针对三种模态在统一检测器中进行系统集成的研究。本文提出了一种三模态框架，利用双流层次视觉Transformer处理RGB、热成像和事件数据。在选定的编码器层深处，我们引入了模态感知门控交换（Modality-Aware Gated Exchange，MAGE）以执行跨传感器通道和空间门控，同时，双向Token交换（Bidirectional Token Exchange，BiTE）模块通过逐深度和逐点精炼机制执行双向Token级注意力，从而生成保持分辨率的融合特征图，用于标准特征金字塔与两阶段检测器。我们构建了一个包含10,489帧的无人机数据集，具有同步且预配准的RGB-热成像-事件流，并标注了24,223个白天及夜间飞行中的车辆实例。通过61组控制消融实验，我们评估了融合层位置、机制（基线MAGE+BiTE、CSSA、GAFF）、模态组合及主干网络容量。实验结果表明，三模态融合在所有双模态基线上均有提升，且融合深度具有显著影响；轻量级CSSA变体以极低的计算成本恢复了大部分性能收益。本研究首次提供了一个系统的基准测试与模块化主干网络，用于三模态无人机目标检测。

## Abstract
Reliable UAV object detection requires robustness to illumination changes, motion blur, and scene dynamics that suppress RGB cues. Thermal long-wave infrared (LWIR) sensing preserves contrast in low light, and event cameras retain microsecond-level temporal edges, but integrating all three modalities in a unified detector has not been systematically studied. We present a tri-modal framework that processes RGB, thermal, and event data with a dual-stream hierarchical vision transformer. At selected encoder depths, a Modality-Aware Gated Exchange (MAGE) applies inter-sensor channel and spatial gating, and a Bidirectional Token Exchange (BiTE) module performs bidirectional token-level attention with depthwise-pointwise refinement, producing resolution-preserving fused maps for a standard feature pyramid and two-stage detector.   We introduce a 10,489-frame UAV dataset with synchronized and pre-aligned RGB-thermal-event streams and 24,223 annotated vehicles across day and night flights. Through 61 controlled ablations, we evaluate fusion placement, mechanism (baseline MAGE+BiTE, CSSA, GAFF), modality subsets, and backbone capacity. Tri-modal fusion improves over all dual-modal baselines, with fusion depth having a significant effect and a lightweight CSSA variant recovering most of the benefit at minimal cost. This work provides the first systematic benchmark and modular backbone for tri-modal UAV-based object detection.

- **研究意义**：首次提出系统性三模态 UAV 检测框架及公开数据集，构建对比可复现的三模态融合基准，为后续研究提供结构化的实验平台。

## 二、方法论：三模态 Transformer 融合框架

- **总体架构**：  
  基于双流层级 Vision Transformer（MiT 模型结构），分为：
  - RGB 主流；
  - 辅助流（Thermal + Event），在指定深度插入融合模块进行特征交互。
  - 输出后接标准 **Feature Pyramid Network (FPN)** 与 **Faster R-CNN 两阶段检测头**。
  
- **关键模块**：
  1. **MAGE（Modality-Aware Gated Exchange）模态感知门控交换**
     - 跨模态的通道与空间门控机制：
       - **通道门控**：通过全局平均与最大池化生成跨模态加权系数；
       - **空间门控**：预测像素级掩码，决定在何处允许跨模态信息流动；
       - 保持每个模态的主路径不变，仅调制跨模态残差项。
  2. **BiTE（Bidirectional Token Exchange）双向Token交换**
     - 双向跨模态注意力机制：
       - 以 Query-Key-Value 跨模态交互方式实现 token 级双向关注；
       - 通过深/点卷积压缩特征，保持空间分辨率；
       - 输出融合后的特征图供后续FPN使用。
  3. **可替换的轻量或高容量变体**
     - **CSSA**（Channel Switching + Spatial Attention）：轻量化版本，通过通道替换 + 空间门控实现低计算融合；
     - **GAFF**（Guided Attentive Feature Fusion）：SE机制引导的高容量融合模块，用于探究更复杂的交互方式。
     
- **公式描述（文字化）**：
  - MAGE模块通过卷积和Sigmoid函数生成双向门控权重，实现跨模态特征的线性组合；
  - BiTE模块基于Transformer的Cross-Attention机制，通过对互为Query/Key/Value的特征集进行双向更新，实现双流互补。

## 六、主要结论与发现

- 三模态融合（RGB+Thermal+Event）在所有双模态组合上表现最优；
- **Fusion Depth** 对性能影响显著：
  - 早期轻量融合（CSSA-s1）性价比最高；

- 使用全日夜数据训练的模型泛化能力最强；
- 数据集及框架提供了可复现的 UAV 三模态检测基准。
