## 摘要
近年来，计算机视觉技术取得了显著进展，并在医学领域中获得了广泛且有影响力的应用。本文提出了一种用于检测巴塞斯达细胞（Bethesda cells）的全新框架，针对与国际生物医学成像研讨会（ISBI）联合举办的 Riva 细胞学挑战赛 Track B 而开发。该研究致力于提升细胞检测中的计算机视觉模型性能，并通过 mAP50-95 指标进行评估。我们提出的解决方案由 YOLO 与 U-Net 架构的集成模型组成，随后采用重叠去除技术与二分类器进行结果优化。在比赛中，我们的框架以 mAP50-95 得分 0.5909 获得第二名。实现与源代码可在以下存储库中获取：github.com/martinamster/riva-trackb。

## Abstract
Computer vision techniques have advanced significantly in recent years, finding diverse and impactful applications within the medical field. In this paper, we introduce a new framework for the detection of Bethesda cells in Pap smear images, developed for Track B of the Riva Cytology Challenge held in association with the International Symposium on Biomedical Imaging (ISBI). This work focuses on enhancing computer vision models for cell detection, with performance evaluated using the mAP50-95 metric. We propose a solution based on an ensemble of YOLO and U-Net architectures, followed by a refinement stage utilizing overlap removal techniques and a binary classifier. Our framework achieved second place with a mAP50-95 score of 0.5909 in the competition. The implementation and source code are available at the following repository: github.com/martinamster/riva-trackb

## 二、方法论与技术流程

### 2.1 核心思想
- 论文提出一种 **多阶段优化检测流程（Multi-stage pipeline）**：
  1. **YOLOv8 模型检测阶段**
  2. **U-Net 语义回归辅助检测**
  3. **多模型集成与后处理优化**
  4. **二分类器精修阶段**

### 2.2 关键技术细节
- **预处理阶段**：
  - 处理 RIVA 数据集中统一的 100×100 像素标注框，重新调整为 10×10–120×120 的多尺度框，以适应细胞形态差异。
  - 在训练中保留标注中心点，调整长宽；推理阶段再恢复标准框大小。
  
- **YOLOv8 模型（Nano / Small / Medium）训练**：
  - 不同尺度（如 20×20 与 50×50）用于捕获不同上下文信息。
  - 训练参数：1024×1024 输入，batch size 16，训练 150 epoch。
  - 高召回率优先——因为漏检比误检代价更大。

- **集成策略（Ensemble）**：
  - 定义两个预测中心之间的最大欧式距离阈值（12像素）判断是否同一个细胞。
  - 同时命中的预测取平均坐标与置信度；单独命中的预测需置信度高于阈值（0.35）方可保留。
  - 第二层集成：将 YOLO 集成结果与 U-Net 输出再合并，统一距离阈值为 12 像素，置信度阈值设为 0。

- **U-Net 热图回归模型**：
  - 使用 **ResNet34** 作为主干网络，预训练于 ImageNet。
  - 将每个细胞中心渲染为二维高斯分布，σ 为目标框大小的 1/6。
  - 网络以 MSE 损失优化，学习生成细胞热图。
  - 测试阶段多尺度推理（0.8×、1.0×、1.2×），并通过局部极值检测提取细胞中心。

- **三阶段优化流程**：
  1. **非极大值抑制（NMS）**：以 IoU=0.75 过滤重叠预测，确保密集细胞不被合并。
  2. **空间密度过滤（Spatial Density Filtering）**：
     - 将图像划为 4×4 栅格。
     - 高密度区（≥30 检测）：置信阈值设为 0.1。
     - 低密度区（<30 检测）：置信阈值放宽至 0.001。
  3. **二分类精修阶段**：
     - 使用 **EfficientNet-B0** 二分类模型区分真实细胞与伪检测噪声。
     - 硬负样本挖掘：与任意真值 IoU < 0.1 的预测标记为“垃圾”。
     - 使用加权二元交叉熵以平衡正负样本。

### 2.3 性能评估指标
- 采用标准检测指标：TP、FP、Recall、Precision、F1。
- 主赛指标为 **mAP50-95**（平均精度在 IoU 0.5 至 0.95 的加权平均）。
