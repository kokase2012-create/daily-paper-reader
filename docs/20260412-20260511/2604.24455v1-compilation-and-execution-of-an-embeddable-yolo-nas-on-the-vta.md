## Abstract
Deploying complex Convolutional Neural Networks (CNNs) on FPGA-based accelerators is a promising way forward for safety-critical domains such as aeronautics. In a previous work, we have explored the Versatile Tensor Accelerator (VTA) and showed its suitability for avionic applications. For that, we developed an initial stand-alone compiler designed with certification in mind. However, this compiler still suffers from some limitations that are overcome in this paper. The contributions consist in extending and fully automating the VTA compilation chain to allow complete CNN compilation and support larger CNNs (which parameters do not fit in the on-chip memory). The effectiveness is demonstrated by the successful compilation and simulated execution of a YOLO-NAS object detection model.

## 二、方法论与技术方案

### 2.1 核心思想
- 构建一个**扩展版、全自动化的 VTA 编译链**，使其具备：
  - **从 ONNX 到 VTA 二进制代码**的自动转换；
  - 对**大规模 CNN 层的分块与分区编译**；
  - 支持在 **CPU 与 VTA 协同执行**条件下完整网络推理。

### 2.2 编译链结构与流程
1. **前端转换（IR 生成）**：  
   从量化后的 ONNX 模型中解析网络拓扑、层参数（如权重、偏置），转换到基于矩阵运算的 **VTA Intermediate Representation（VTA IR）**。
2. **中间表示（VTA IR）**：  
   采用 JSON 格式描述，每一层包括矩阵定义、LOAD/GEMM/ALU/STORE 操作序列。  
   - GEMM 表示矩阵乘积；
   - ALU 表示逐元素操作（如 MAX、ADD、MUL）；
   - STORE/LOAD 负责在 DRAM 与片上存储间交换数据。
3. **层级分解策略**：
   - 采用 **块矩阵分解 (block matrix decomposition)** 和 **矩阵分区 (matrix partitioning)** 两级分解：
     - **块化 (blocking)**：将大矩阵拆分为固定大小的 `bs×bs` 子块；
     - **分区 (partitioning)**：当子块仍超出片上容量时，进一步划分并多次加载执行。
   - 提出四种启发式矩阵分区策略（见 Figure 8），通过不同计算顺序优化访存与累积操作。
4. **CPU-VTA 协作机制**：  
   - CPU 负责控制指令加载、层间数据重整、无法量化或浮点操作（如加法、反量化等）；
   - VTA 专注于量化矩阵运算加速（int8 → int32 计算）。
5. **自动化执行链生成**：  
   编译器生成三类产物：
   - 各层对应的 `.bin` 二进制指令和权重；
   - CPU 参数文件（协调层连接与内存映射）；
   - 函数级仿真输入配置，可直接运行在 C++ 功能级模拟器中。

## 六、主要结论与发现

1. **实现成果**：  
   成功将量化 YOLO-NAS 模型自动编译为 VTA 可执行二进制文件，实现完整的层间数据流与控制链。
2. **结果正确性**：  
   仿真结果与基于 Numpy 的 ONNX 数学基准一致，数值误差控制在 ±1 范围内。
3. **性能观察**：
   - 生成指令总量约 1,076 万条，UOP（微操作）约 912 万；
   - 分区策略影响指令数量高达 ±40%，但对 UOP 数量无影响；
   - 编译模型的内存占用中，偏置矩阵扩展为主要开销（从向量扩张为矩阵）。
4. **方法意义**：  
   建立了可认证的 FPGA CNN 自动编译链，为航空级人工智能部署提供开源工具基础。
