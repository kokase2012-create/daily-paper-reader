## 摘要
海洋机器人的海洋垃圾检测对于生态保护至关重要，但由于图像质量较低、模糊、背景复杂以及目标较小，检测性能常常受限。为应对这些挑战，我们提出了一种基于 YOLO 的增强检测框架 YOLO-MD。该框架设计了一个双分支卷积增强自注意力（Dual-Branch Convolutional Enhanced Self-Attention, DB-CASA）模块，用于加强空间与通道间的交互，从而改善退化图像中的特征表示。此外，引入了一种轻量级的基于位移的操作，用以提升不同尺度目标的细粒度特征提取效果，同时保持参数效率。我们进一步提出了 SFG-Loss，通过动态样本重加权来缓解类别不平衡和优化不稳定问题。在 UODM 数据集上的实验结果表明，YOLO-MD 实现了 0.875 的精确率、0.822 的 F1 分数和 0.849 的 mAP50，性能优于当前最新的先进方法。该方法的有效性还通过实际机器人边缘部署实验得到了验证。

## Abstract
Marine debris detection for ocean robot is crucial for ecological protection, yet performance is often degraded by low-quality images with blur, complex backgrounds, and small targets. To address these challenges, we propose YOLO-MD, an enhanced YOLO-based detection framework. A Dual-Branch Convolutional Enhanced Self-Attention (DB-CASA) module is designed to strengthen spatial-channel interactions, improving feature representation in degraded images. Additionally, a lightweight shift-based operation is introduced to enhance fine-grained feature extraction for objects of varying scales while maintaining parameter efficiency. We further propose SFG-Loss to mitigate class imbalance and optimization instability via dynamic sample reweighting. Experiments on the UODM dataset demonstrate that YOLO-MD achieves 0.875 precision, 0.822 F1-score, and 0.849 mAP50, outperforming the latest state-of-the-art methods. The effectiveness of this method has also been verified through real-world robotic edge deployment experiments.

## 二、方法论：YOLO‑MD 框架设计

YOLO‑MD 是一种 **基于 YOLO 的增强检测体系**，核心创新包括模块改进、特征交互优化与新损失函数设计：

### 1. Dual‑Branch Convolutional Enhanced Self‑Attention (DB‑CASA)
- **思路**：结合卷积与自注意力，增强空间与通道的双向特征交互。
- **结构**：包含两个分支：
  - **空间分支（Spatial branch）**：利用深度可分离卷积提取局部边缘与纹理信息，特别强化小目标的空间可分辨性。
  - **通道分支（Channel branch）**：通过卷积与全局平均池化捕获跨通道依赖，保持细粒度通道表达。
- **轻量化创新**：  
  - 使用加法融合 (Additive fusion) 代替拼接，显著降低计算量。  
  - 避免大规模相似度矩阵计算。
- **核心输出形式**：通过 Q, K, V 线性变换后，融合空间与通道注意力后输入到线性层，实现高效空间‑通道协同建模。

### 2. Feature Shift Fusion Module (FSFM)
- **思路**：利用**无参数的空间偏移操作（shift）**实现跨维度特征交互。
- **机制**：
  - 将输入特征沿通道维拆分为数个组；
  - 对各组施加不同方向（上下左右）的空间偏移；
  - 汇合后进行拼接，实现多方向局部上下文融合；
  - 嵌入至 C3k2 模块结构内，与残差连接共同增强信息流通。
- **优势**：无需额外卷积参数，但改善了小目标区域的局部特征交互。

### 3. SFG‑Loss（Slide‑Focal‑GIoU Loss）
- **目标**：解决类间不平衡与训练不稳定性。
- **构成**：
  - **主项：** 采用 Generalized IoU (GIoU) 提供鲁棒定位监督；
  - **滑动权重项（Slide Weight）**：对困难样本赋予更高权重；
  - **Focaler‑IoU 调制**：截断极端易／难样本，保持优化稳定性；
- **实质效果**：动态重加权机制在保持稳定收敛的同时提升了硬样本检测准确性。
