## 摘要
海洋机器人的海洋垃圾检测对于生态保护至关重要，但由于图像质量较低（包括模糊、复杂背景和小目标等），检测性能往往受限。为应对这些挑战，我们提出了一种基于 YOLO 改进的检测框架——YOLO-MD。该框架设计了一个双分支卷积增强自注意力（Dual-Branch Convolutional Enhanced Self-Attention，DB-CASA）模块，以强化空间与通道间的交互，从而提升退化图像下的特征表示能力。此外，我们引入了一种轻量级基于偏移的操作，以增强不同尺度目标的细粒度特征提取，同时保持参数的高效性。我们进一步提出了 SFG-Loss，通过动态样本重新加权来缓解类别不平衡和优化不稳定性。在 UODM 数据集上的实验表明，YOLO-MD 分别达到了 0.875 的精度、0.822 的 F1 值和 0.849 的 mAP50，优于当前最先进的方法。该方法的有效性还通过实际的机器人边缘部署实验得到了验证。

## Abstract
Marine debris detection for ocean robot is crucial for ecological protection, yet performance is often degraded by low-quality images with blur, complex backgrounds, and small targets. To address these challenges, we propose YOLO-MD, an enhanced YOLO-based detection framework. A Dual-Branch Convolutional Enhanced Self-Attention (DB-CASA) module is designed to strengthen spatial-channel interactions, improving feature representation in degraded images. Additionally, a lightweight shift-based operation is introduced to enhance fine-grained feature extraction for objects of varying scales while maintaining parameter efficiency. We further propose SFG-Loss to mitigate class imbalance and optimization instability via dynamic sample reweighting. Experiments on the UODM dataset demonstrate that YOLO-MD achieves 0.875 precision, 0.822 F1-score, and 0.849 mAP50, outperforming the latest state-of-the-art methods. The effectiveness of this method has also been verified through real-world robotic edge deployment experiments.

## 二、方法论：核心思想与关键技术

### 1. 整体框架思想

YOLO-MD 基于 YOLO 系列检测器，主要改进以下三方面：
- **特征增强**：设计双分支卷积增强自注意力模块（DB-CASA）；
- **局部交互优化**：引入无参数化的特征位移融合模块（FSFM）；
- **训练优化**：提出动态加权的损失函数 SFG-Loss，缓解类别不平衡。

### 2. Dual-Branch Convolutional Enhanced Self-Attention（DB-CASA）

- 将特征图划分为两个分支：
  - **空间交互分支**：利用深度可分离卷积（DWConv）与卷积降维获取空间注意信息；
  - **通道交互分支**：结合全局平均池化与卷积，捕获跨通道依赖；
- 各分支对输入特征生成注意力权重（空间与通道映射），通过**加法融合**保留细粒度特征；
- 与传统注意力不同，DB-CASA 使用轻量加性相似度而非矩阵乘积，降低了计算复杂度。

**核心公式描述（文字说明）：**  
模块先计算空间注意图 `Fs(x)` 与通道注意图 `Fc(x)`，再分别作用于 Query 和 Key，最后经线性变换加权融合生成最终输出，从而实现低成本的空间–通道双重特征增强。

### 3. Feature Shift Fusion Module（FSFM）

- 将输入特征沿通道维分为四组；
- 对不同分组施加**水平和垂直方向位移操作**（不引入新参数）；
- 再将处理后的特征在通道维拼接，实现多方向局部特征交互；
- 集成于 C3k2 结构中，通过残差连接促进特征传播；

该模块相比常规卷积操作 **计算零开销**，提升小目标的边界一致性与位置信息完整性。

### 4. SFG-Loss（Slide–Focaler Generalized IoU Loss）

- 在 GIoU 损失基础上引入动态加权机制，对困难样本赋予更大权重；
- 结合 Focaler-IoU 的截断策略，抑制极端样本（过易或过难）的梯度波动；
- 设计目标为：
  - 增强难样本学习；
  - 防止训练不稳定；
  - 平衡类间差异。  

最终损失融合了定位精度、样本难度与训练稳定性三方面因素。
