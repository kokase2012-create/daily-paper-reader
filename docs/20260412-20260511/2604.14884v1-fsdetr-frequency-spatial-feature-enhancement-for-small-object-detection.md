## 摘要
由于下采样导致的特征退化、密集簇中相互遮挡以及复杂背景干扰，小目标检测仍然是一个重大挑战。针对这些问题，本文提出了基于 RT-DETR 基线的频率-空间特征增强框架 FSDETR。该方法通过建立协同建模机制，有效利用互补的结构信息。具体而言，空间层次注意块（Spatial Hierarchical Attention Block，SHAB）捕捉局部细节与全局依赖，强化语义表示。此外，为减轻密集场景中的遮挡问题，基于可变形注意力的尺度内特征交互模块（Deformable Attention-based Intra-scale Feature Interaction，DA-AIFI）通过动态采样聚焦于信息丰富的区域。最后，频率-空间特征金字塔网络（Frequency-Spatial Feature Pyramid Network，FSFPN）通过跨域频率-空间块（Cross-domain Frequency-Spatial Block，CFSB）融合频率滤波与空间边缘提取，以保留细粒度细节。实验结果表明，FSDETR 仅有 14.7M 参数，在 VisDrone 2019 数据集上取得了 13.9% APS，在 TinyPerson 数据集上取得了 48.95% AP50 tiny，表现出在小目标基准上的强劲性能。代码与模型可在 https://github.com/YT3DVision/FSDETR 获取。

## Abstract
Small object detection remains a significant challenge due to feature degradation from downsampling, mutual occlusion in dense clusters, and complex background interference. To address these issues, this paper proposes FSDETR, a frequency-spatial feature enhancement framework built upon the RT-DETR baseline. By establishing a collaborative modeling mechanism, the method effectively leverages complementary structural information. Specifically, a Spatial Hierarchical Attention Block (SHAB) captures both local details and global dependencies to strengthen semantic representation. Furthermore, to mitigate occlusion in dense scenes, the Deformable Attention-based Intra-scale Feature Interaction (DA-AIFI) focuses on informative regions via dynamic sampling. Finally, the Frequency-Spatial Feature Pyramid Network (FSFPN) integrates frequency filtering with spatial edge extraction via the Cross-domain Frequency-Spatial Block (CFSB) to preserve fine-grained details. Experimental results show that with only 14.7M parameters, FSDETR achieves 13.9% APS on VisDrone 2019 and 48.95% AP50 tiny on TinyPerson, showing strong performance on small-object benchmarks. The code and models are available at https://github.com/YT3DVision/FSDETR.

## 二、方法论与核心技术

### 1. 总体架构设计

- 构建于 **RT-DETR**（Real-Time Detection Transformer）基线之上；
- 设计三大关键创新组件：
  - 空间层次注意力（**SHAB**）
  - 可变形尺度内特征交互（**DA-AIFI**）
  - 频率-空间特征金字塔网络（**FSFPN**）及其核心块（**CFSB**）

### 2. 模块细节与原理

#### （1）Spatial Hierarchical Attention Block — SHAB
- 作用：缓解深层下采样引起的特征侵蚀；
- 实现思路：
  - 在 CSPNet 主干网络的 C2f 结构中嵌入轻量级自注意力；
  - 通道分割策略（比例 r=0.5）：一半通道参与自注意力，全局建模；另一半保持恒等映射；
  - 在提升全局依赖的同时保留局部细节；
- 优点：低计算开销下扩大感受野，改善小目标的上下文依赖。

#### （2）Deformable Attention-based Intra-scale Feature Interaction — DA-AIFI
- 作用：提升密集场景中几何对齐与定位精度；
- 关键机制：
  - 可变形注意力从参考点周围动态采样 K 个位置；
  - 计算复杂度由 \( O(N^2) \) 降至 \( O(NK) \)，其中 K≪N；
  - 模块对噪声敏感区域有自适应抑制；
- 本质：通过“稀疏注意力 + 动态偏移”聚焦信息丰富区域。

#### （3）Frequency-Spatial Feature Pyramid Network — FSFPN
- 思想：将传统FPN视为**信号恢复与增强**过程；
- 特征融合路径：
  1. 底层采用 **SPDConv** 替代步幅卷积以维持空间解析度；
  2. 顶层使用 **Soft Nearest Neighbor Interpolation** 实现纹理友好型上采样；
  3. 中心模块为 **Cross-domain Frequency-Spatial Block（CFSB）**：
     - **空间分支**：施加 Scharr 算子提取边缘梯度，融合水平与垂直方向；
     - **频率分支**：通过 2D DFT / IDFT 转换与可学习滤波掩码强化高频信息；
     - 最终融合公式：  
       \( X_{out} = Conv(X_{spatial} + X_{freq}) \)
     - 输出既包含精确边缘也保留纹理信号。

#### （4）训练损失优化
- 总损失：  
  \( L = \lambda_{cls}L_{VFL} + \lambda_{L1}L_{L1} + \lambda_{IoU}L_{Focaler-EIoU} \)
- 特征：
  - **Varifocal Loss**：引入 IoU 权重提升高质量正样本贡献；
  - **Focaler-EIoU Loss**：针对低重叠框重建梯度；
  - 多任务加权优化，使模型同时具备分类稳健性与几何精准度。
