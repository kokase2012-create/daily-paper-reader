## 摘要
基于 Transformer 的检测器在小目标检测方面取得了进展，但它们常常效率不高，且易受背景干扰导致的查询噪声影响，这迫使深度解码器去细化低质量的查询。我们提出了 HELP（Heatmap-guided Embedding Learning Paradigm，热力图引导的嵌入学习范式），这是一个噪声感知的位置信息与语义融合框架，用于研究在何处嵌入位置信息——通过选择性地在前景显著区域保留位置编码，同时抑制背景杂波。在 HELP 中，我们提出了 Heatmap-guided Positional Embedding（HPE，热力图引导的位置嵌入）作为核心嵌入机制，并通过 heatbar 可视化进行可解释诊断与微调。HPE 被集成到编码器和解码器中：它通过注入热力图感知的位置编码来指导噪声抑制的特征编码，并在解码前通过基于梯度的掩膜过滤器筛选以背景为主的嵌入，从而实现高质量查询检索。为解决复杂小目标中的特征稀疏问题，我们引入 Linear-Snake Convolution 以丰富与检索相关的特征表示。基于梯度的热力图监督仅在训练阶段使用，推理时无需额外的梯度计算。结果表明，我们的设计将解码器层数从八层减少至三层，实现了 59.4% 的参数减少（66.3M 对比 163M），同时在较低计算预算下仍在多个基准上保持了稳定的准确率提升。代码仓库：https://github.com/yidimopozhibai/Noise-Suppressed-Query-Retrieval

## Abstract
Transformer-based detectors have advanced small-object detection, but they often remain inefficient and vulnerable to background-induced query noise, which motivates deep decoders to refine low-quality queries. We present HELP (Heatmap-guided Embedding Learning Paradigm), a noise-aware positional-semantic fusion framework that studies where to embed positional information by selectively preserving positional encodings in foreground-salient regions while suppressing background clutter. Within HELP, we introduce Heatmap-guided Positional Embedding (HPE) as the core embedding mechanism and visualize it with a heatbar for interpretable diagnosis and fine-tuning. HPE is integrated into both the encoder and decoder: it guides noise-suppressed feature encoding by injecting heatmap-aware positional encoding, and it enables high-quality query retrieval by filtering background-dominant embeddings via a gradient-based mask filter before decoding. To address feature sparsity in complex small targets, we integrate Linear-Snake Convolution to enrich retrieval-relevant representations. The gradient-based heatmap supervision is used during training only, incurring no additional gradient computation at inference. As a result, our design reduces decoder layers from eight to three and achieves a 59.4% parameter reduction (66.3M vs. 163M) while maintaining consistent accuracy gains under a reduced compute budget across benchmarks. ## 2. 方法论与技术细节

### 2.1 框架总体结构：HELP
HELP（Heatmap-guided Embedding Learning Paradigm）是一个基于噪声感知的嵌入学习框架，核心思想为——**“学习在何处嵌入位置编码”**。  
其主要设计包含三个模块：

1. **HPE (Heatmap-guided Positional Embedding)**  
   - 利用梯度导出的热力图（heatmap）估计前景显著性，将位置嵌入仅保留在高响应区域。  
   - 背景区域的位置信息被掩蔽（mask），从而减少位置噪声传入查询层。  
   - 训练时计算二阶与三阶梯度生成热力图，推理阶段无需梯度计算，保证推理效率。  
   - 核心公式示意：
     - 基于梯度的重要性权重计算类热力图：  
       \(H_{class} = ReLU( \sum_k \beta_{ck} A_k(i,j) )\)
     - 结合边框几何热力图：  
       \(H_{mixed} = λH_{class} + (1−λ)H_{bbox}\)
     - 二值掩膜确定前景位置嵌入：
       \[
       Mask(i,j)=
       \begin{cases}
       1, & H_{map}(i,j) > \tau\\
       0, & 其他
       \end{cases}
       \]

2. **MOHFE (Multi-Scale ObjectBox-Heatmap Fusion Encoder)**  
   - 在编码器阶段融合类别与几何热力图，将语义与位置联合嵌入。  
   - 通过多尺度特征融合构建噪声抑制的编码特征（K、V）。

3. **HQ-Retrieval (High-Quality Query Retrieval)**  
   - 基于热力图权重生成高质量解码查询（Q），并在解码前滤除背景主导的嵌入。
   - 采用可变形注意力 (Deformable Attention) 执行高精度解码。

4. **LSConv (Linear-Snake Convolution)**  
   - 为解决小目标特征稀疏问题而设计。
   - 包含两条卷积分支：  
     - 线性分支捕获稳定的直线结构；  
     - “蛇形”分支通过可学习偏移捕获弯曲或碎片化特征。  
   - 相较于标准与可变形卷积，LSConv 提供更平衡的几何一致性与灵活感知力。
