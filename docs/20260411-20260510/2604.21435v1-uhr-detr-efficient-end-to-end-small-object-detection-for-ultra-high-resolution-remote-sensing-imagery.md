## 摘要
超高分辨率（UHR）影像已成为现代遥感的重要组成部分，提供了前所未有的空间覆盖能力。然而，在如此巨大的场景中进行小目标检测面临一个关键困境：为了保留小目标的原始分辨率会导致难以承受的内存瓶颈；相反，常规的折中方法，如图像下采样或图块裁剪，要么使小目标丢失，要么破坏上下文信息。为打破这一困境，我们提出了UHR-DETR，这是一种面向UHR影像的高效端到端Transformer检测器。首先，我们引入了Coverage-Maximizing Sparse Encoder（覆盖最大化稀疏编码器），该模块动态分配有限的计算资源给信息量丰富的高分辨率区域，以在最小空间冗余下实现最大目标覆盖。其次，我们设计了Global-Local Decoupled Decoder（全局-局部分离解码器）。通过结合宏观场景感知与微观目标细节，该模块能够消除语义歧义，防止场景碎片化。在UHR影像数据集（如STAR和SODA-A）上的大量实验表明，在严格的硬件条件下（如单张24GB RTX 3090显卡），UHR-DETR表现出卓越的性能。与标准滑动窗口基线方法相比，它在STAR数据集上实现了2.8%的mAP提升，并带来了10倍的推理速度提升。我们的代码和模型将在 https://github.com/Li-JingFang/UHR-DETR 提供。

## Abstract
Ultra-High-Resolution (UHR) imagery has become essential for modern remote sensing, offering unprecedented spatial coverage. However, detecting small objects in such vast scenes presents a critical dilemma: retaining the original resolution for small objects causes prohibitive memory bottlenecks. Conversely, conventional compromises like image downsampling or patch cropping either erase small objects or destroy context. To break this dilemma, we propose UHR-DETR, an efficient end-to-end transformer-based detector designed for UHR imagery. First, we introduce a Coverage-Maximizing Sparse Encoder that dynamically allocates finite computational resources to informative high-resolution regions, ensuring maximum object coverage with minimal spatial redundancy. Second, we design a Global-Local Decoupled Decoder. By integrating macroscopic scene awareness with microscopic object details, this module resolves semantic ambiguities and prevents scene fragmentation. Extensive experiments on the UHR imagery datasets (e.g., STAR and SODA-A) demonstrate the superiority of UHR-DETR under strict hardware constraints (e.g., a single 24GB RTX 3090). It achieves a 2.8\% mAP improvement while delivering a 10$\times$ inference speedup compared to standard sliding-window baselines on the STAR dataset. Our codes and models will be available at https://github.com/Li-JingFang/UHR-DETR.

## 二、方法论与技术框架

### 核心思想
论文提出 **UHR-DETR**（Efficient End-to-End Small Object Detector for UHR Imagery），通过 **“全局引导稀疏局部感知”** 的思想，实现端到端的小目标检测。  
其核心创新包括两个模块：

1. **Coverage-Maximizing Sparse Encoder（覆盖最大化稀疏编码器）**
   - 模仿人类视觉的“凝视机制”：先粗看全局，再聚焦关键区域。
   - 利用轻量级全局特征提取器生成 **Gain Map（增益图）**，用于评估每个子区域的“目标信息价值”。
   - 在此基础上，使用 **迭代软减贪心算法（Iterative Soft-Subtraction Greedy Algorithm, ISSGA）** 选择最有价值的高分辨率图块。
   - 将图块选择转化为**几何最大覆盖问题**（Maximum Coverage Problem），在有限 patch 预算 K 下最大化目标覆盖率。
   - 公式思想：
     - Gain Map 基于预测各子区域目标覆盖率的分类期望值；
     - ISSGA 迭代选取最大增益区域，通过线性衰减核防止冗余重叠；
     - 输出稀疏的高分辨率特征集合。

2. **Global–Local Decoupled Decoder（全局-局部解耦解码器）**
   - 分为两个阶段：
     - **Global Macro-Interaction**：通过多头跨注意力在全局特征与查询之间注入场景级语义；
     - **Local Micro-Interaction**：采用多尺度可变形注意力在高分辨率局部特征内进行细粒度检测。
   - 这种层级式交互可将**全局上下文约束与局部精细特征结合**，提升小目标语义辨识能力。

### 损失函数组成
整体优化目标结合三部分：
\[
L_{total} = \lambda_{map}L_{map} + \lambda_{LPM}L_{LPM} + L_{det}
\]
- Gain Map 的分布焦点损失（Distribution Focal Loss）；
- 辅助局部峰值边距损失（Local Peak Margin Loss, LPM），用于增强对象聚集区域的凸显；
- 检测部分采用 Hungarian 匹配，包含分类的 Varifocal Loss 与边框的 L1 + GIoU 损失。
