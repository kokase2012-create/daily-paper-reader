## 摘要
可靠的无人机（UAV）目标检测需要对光照变化、运动模糊以及场景动态具有鲁棒性，这些因素常常削弱RGB线索。热成像长波红外（LWIR）传感在弱光条件下能够保持对比度，事件相机能够保留微秒级的时间边缘，但将这三种模态整合到一个统一的检测器中尚未得到系统研究。我们提出了一种三模态框架，利用双流层次视觉Transformer来处理RGB、热成像和事件数据。在选定的编码器深度处，模态感知门控交换（MAGE）模块执行传感器间的通道和空间门控，而双向令牌交换（BiTE）模块通过深度-点卷积细化实现双向令牌级注意力，从而生成分辨率保持的融合特征图，供标准特征金字塔和两阶段检测器使用。我们引入了一个包含10,489帧的无人机数据集，具有同步且预对齐的RGB-热成像-事件流，以及在昼夜飞行中标注的24,223个车辆。通过61组控制消融实验，我们评估了融合位置、机制（基线MAGE+BiTE、CSSA、GAFF）、模态子集以及主干容量。三模态融合在所有双模态基线之上都有改进，其中融合深度具有显著影响，而轻量级的CSSA变体以最小代价恢复了大部分收益。本研究为基于无人机的三模态目标检测提供了首个系统基准和模块化主干网络。

## Abstract
Reliable UAV object detection requires robustness to illumination changes, motion blur, and scene dynamics that suppress RGB cues. Thermal long-wave infrared (LWIR) sensing preserves contrast in low light, and event cameras retain microsecond-level temporal edges, but integrating all three modalities in a unified detector has not been systematically studied. We present a tri-modal framework that processes RGB, thermal, and event data with a dual-stream hierarchical vision transformer. At selected encoder depths, a Modality-Aware Gated Exchange (MAGE) applies inter-sensor channel and spatial gating, and a Bidirectional Token Exchange (BiTE) module performs bidirectional token-level attention with depthwise-pointwise refinement, producing resolution-preserving fused maps for a standard feature pyramid and two-stage detector.   We introduce a 10,489-frame UAV dataset with synchronized and pre-aligned RGB-thermal-event streams and 24,223 annotated vehicles across day and night flights. Through 61 controlled ablations, we evaluate fusion placement, mechanism (baseline MAGE+BiTE, CSSA, GAFF), modality subsets, and backbone capacity. Tri-modal fusion improves over all dual-modal baselines, with fusion depth having a significant effect and a lightweight CSSA variant recovering most of the benefit at minimal cost. This work provides the first systematic benchmark and modular backbone for tri-modal UAV-based object detection.

## 一、研究背景与核心问题

- **研究动机**：无人机在复杂环境下（如低光照、运动模糊、快速场景变化）执行目标检测任务时，传统的RGB相机信息往往失效。

- **研究目标**：建立一个完备的、可扩展的三模态融合Transformer框架，系统探索不同模态交互方式、融合深度及机制对无人机目标检测性能的影响，同时构建同步的三模态数据集以推动公平比较与基准建立。

## 二、方法论与模型结构

### 1. 整体思路  
论文提出一个**基于层次化Vision Transformer（MiT）的双流网络架构**，以实现三模态（RGB、热像、事件）的有效融合。  
- RGB通道为主流，热像+事件通道组成辅助流；
- 双流Transformer在不同层级设置“融合钩子”，按选定深度交互；
- 输出特征进入统一的FPN金字塔与标准两阶段检测头（Faster R-CNN）。

### 2. 核心模块  
**（1）MAGE：Modality-Aware Gated Exchange**  
- 作用：实现跨模态的**通道与空间门控**。  
- 思想：根据两个模态的联合表示，学习方向性门控权重，调节信息流动：
  - 通道门控由全局平均池化与MLP生成；
  - 空间门控预测像素级掩码；
  - 仅调制交叉残差信号，保留各模态自特征路径，避免信息污染。

**（2）BiTE：Bidirectional Token Exchange**  
- 作用：进行双向的令牌级注意力融合。  
- 机制：
  - 将两个模态特征映射为token序列；
  - 使用跨注意力计算相互更新；
  - 更新结果经深度卷积与1×1投影还原空间与通道尺寸；
  - 最终生成保持分辨率的融合特征。

### 3. 替代融合机制  
- **CSSA（Channel Switching + Spatial Attention）**：轻量化设计，通过通道评分与跨模态替换再配合空间选择，实现快速早期融合。  
- **GAFF（Guided Attentive Feature Fusion）**：更高容量方案，利用SE通道校准与空间引导残差更新，适合深层次融合。

### 4. 特征流转  
全流程为：
1. 输入五通道张量 → 双流MiT编码；
2. 选定阶段执行MAGE+BiTE或替代模块；
3. 融合特征送入FPN；
4. 采用Faster R-CNN检测头进行边界框与类别预测。

## 六、主要结论与发现

- **三模态优于双模态**：RGB+热像+事件表现最佳（mAP 84.24%），显著高于任意双模态组合。  
- **融合层级关键**：不同深度对性能影响显著，深层单次融合优于多层频繁融合。  
- **MAGE+BiTE最优**：比BiTE或MAGE单独使用分别高约3–7个百分点。  
- **轻量CSSA高效**：在早期单阶段融合下仅略低于基线，但计算代价更小。  
- **GAFF稳健但不超越**：在深层阶段有竞争力，但参数量更高。  
- **模型容量呈非单调趋势**：MiT-B1在参数60M时性能最优；更大模型因过拟合反而下降。  
- **昼夜混合训练最稳健**：单一照明条件训练模型无法泛化至另一条件。
