## 摘要
可靠的无人机（UAV）目标检测需要在光照变化、运动模糊以及削弱RGB线索的场景动态下保持鲁棒性。热成像长波红外（LWIR）传感在弱光条件下可保持对比度，而事件相机则以微秒级时间精度保留边缘信息，但将这三种模态集成于统一检测器中尚未被系统研究。我们提出了一种三模态框架，使用双流层次视觉Transformer处理RGB、热成像和事件数据。在选定的编码器深度处，模态感知门控交换（Modality-Aware Gated Exchange, MAGE）执行跨传感器通道与空间门控，双向Token交换（Bidirectional Token Exchange, BiTE）模块通过逐层点式精炼执行双向Token级注意力，从而生成保持分辨率的融合特征图，供标准特征金字塔与两阶段检测器使用。我们构建了一个包含10,489帧的无人机数据集，具有同步且预对齐的RGB-热成像-事件流，并在昼夜飞行中标注了24,223个车辆目标。通过61组对照消融实验，我们评估了融合位置、机制（基线MAGE+BiTE、CSSA、GAFF）、模态子集及主干容量。三模态融合在所有双模态基线之上均取得提升，融合深度具有显著影响，而轻量级CSSA变体以极低代价恢复了大部分性能收益。本研究提供了首个系统化基准与模块化骨干网络，用于三模态无人机目标检测。

## Abstract
Reliable UAV object detection requires robustness to illumination changes, motion blur, and scene dynamics that suppress RGB cues. Thermal long-wave infrared (LWIR) sensing preserves contrast in low light, and event cameras retain microsecond-level temporal edges, but integrating all three modalities in a unified detector has not been systematically studied. We present a tri-modal framework that processes RGB, thermal, and event data with a dual-stream hierarchical vision transformer. At selected encoder depths, a Modality-Aware Gated Exchange (MAGE) applies inter-sensor channel and spatial gating, and a Bidirectional Token Exchange (BiTE) module performs bidirectional token-level attention with depthwise-pointwise refinement, producing resolution-preserving fused maps for a standard feature pyramid and two-stage detector.   We introduce a 10,489-frame UAV dataset with synchronized and pre-aligned RGB-thermal-event streams and 24,223 annotated vehicles across day and night flights. Through 61 controlled ablations, we evaluate fusion placement, mechanism (baseline MAGE+BiTE, CSSA, GAFF), modality subsets, and backbone capacity. Tri-modal fusion improves over all dual-modal baselines, with fusion depth having a significant effect and a lightweight CSSA variant recovering most of the benefit at minimal cost. This work provides the first systematic benchmark and modular backbone for tri-modal UAV-based object detection.

## 二、方法论：三模态融合 Transformer 框架  

### 1. 整体架构思想  
- 采用 **双流层次 Transformer（MiT 结构）** ：一个流处理 RGB 图像，另一个流处理 **热红外+事件（TE）** 数据，减少冗余计算。  
- 在编码器的不同深度插入融合模块，实现跨模态信息的层次化交互。  
- 检测头采用标准 **Feature Pyramid Network（FPN）+ Faster R-CNN 两阶段检测器**。  

### 2. 核心模块机制  

#### （1）模态感知门控交换（MAGE）  
- **功能**：在通道与空间两个层面执行跨模态增强。  
- **实现逻辑**：  
  - 对 RGB 与 TE 的特征进行拼接，利用全局平均与最大池化生成通道权重。  
  - 通过 1×1 MLP + Sigmoid 形成两方向的通道门控 `(w_c_TE→RGB, w_c_RGB→TE)`。  
  - 同时预测空间掩膜 `(w_s_TE→RGB, w_s_RGB→TE)` 控制跨模态传递的区域。  
  - 最终输出：
    ```
    x̂_rgb = x_rgb + w_s_TE→RGB · w_c_TE→RGB · x_TE
    x̂_TE  = x_TE  + w_s_RGB→TE · w_c_RGB→TE · x_rgb
    ```
  - 在保持自身流的特征路径不变的同时实现跨模态强化。

#### （2）双向 Token 交换（BiTE）  
- **功能**：通过交叉注意力机制使两模态的特征在 token 层面相互强化。  
- **实现步骤**：
  - 对两流特征映射展平为 Token 序列。  
  - 分别计算 Query、Key、Value，并执行 **双向 Cross Attention**。  
  - 使用深度卷积恢复局部性，再用 1×1 卷积压缩通道得到融合输出。  
- **特点**：保持分辨率与通道宽度不变，可灵活插入任意编码层。  

### 3. 替代与对比机制  
- **CSSA**（Channel Switching + Spatial Attention）：轻量版本，以通道阈值替换方式进行快速融合。  
- **GAFF**（Guided Attentive Feature Fusion）：高容量版本，结合 Squeeze-Excitation 与空间引导残差更新。  
- 这些机制在相同架构下替代 MAGE+BiTE 模块进行公平对比。

## 六、主要结论与发现  

- 三模态融合（RGB+Thermal+Event）在所有双模态及单模态基线下 **均表现最优**。  
  - MiT-B1 主干下达到 **84.24% mAP**。  
- **融合深度影响显著**：早层融合适合轻量结构（CSSA），深层融合更适合高容量模块（GAFF）。  
- **事件模态贡献**：在低光和高速场景下提升检测鲁棒性，尤其在夜间 **减少误检与补全漏检车辆**。  
- **融合模块效果**：MAGE-only（81.01%）与 BiTE-only（76.88%）均明显低于合并版（84.24%）。  
- **多光照训练重要性**：在日夜数据联合训练下性能最稳健（82.24%），单一光照训练则呈现过拟合。
