## 摘要
由于下采样导致的特征退化、密集簇中的相互遮挡以及复杂背景干扰，小目标检测仍然是一个重大的挑战。为了解决这些问题，本文提出了一种基于 RT-DETR 基线的频率-空间特征增强框架 FSDETR。通过建立协同建模机制，该方法有效利用了互补的结构信息。具体而言，空间层次注意力块（Spatial Hierarchical Attention Block, SHAB）同时捕获局部细节与全局依赖关系，以增强语义表征。此外，为了缓解密集场景中的遮挡问题，基于可变形注意力的尺度内特征交互模块（Deformable Attention-based Intra-scale Feature Interaction, DA-AIFI）通过动态采样聚焦于信息丰富的区域。最后，频率-空间特征金字塔网络（Frequency-Spatial Feature Pyramid Network, FSFPN）通过跨域频率-空间块（Cross-domain Frequency-Spatial Block, CFSB）将频域滤波与空间边缘提取相结合，以保留细粒度的细节。实验结果表明，FSDETR 仅含 14.7M 参数，在 VisDrone 2019 数据集上达到 13.9% 的 APS，在 TinyPerson 数据集上实现 48.95% 的 AP50 tiny，展示了在小目标检测基准上的优异性能。代码与模型可在 https://github.com/YT3DVision/FSDETR 获取。

## Abstract
Small object detection remains a significant challenge due to feature degradation from downsampling, mutual occlusion in dense clusters, and complex background interference. To address these issues, this paper proposes FSDETR, a frequency-spatial feature enhancement framework built upon the RT-DETR baseline. By establishing a collaborative modeling mechanism, the method effectively leverages complementary structural information. Specifically, a Spatial Hierarchical Attention Block (SHAB) captures both local details and global dependencies to strengthen semantic representation. Furthermore, to mitigate occlusion in dense scenes, the Deformable Attention-based Intra-scale Feature Interaction (DA-AIFI) focuses on informative regions via dynamic sampling. Finally, the Frequency-Spatial Feature Pyramid Network (FSFPN) integrates frequency filtering with spatial edge extraction via the Cross-domain Frequency-Spatial Block (CFSB) to preserve fine-grained details. Experimental results show that with only 14.7M parameters, FSDETR achieves 13.9% APS on VisDrone 2019 and 48.95% AP50 tiny on TinyPerson, showing strong performance on small-object benchmarks. The code and models are available at https://github.com/YT3DVision/FSDETR.

## 二、方法论与技术设计

### 1. 整体架构

FSDETR 保留 RT-DETR 的端到端检测范式，同时在特征提取与多尺度融合环节做针对性增强，包括：
- **Backbone**：改进的 CSPNet 结构，输出多尺度特征 {P2, P3, P4, P5}。
- **Encoder**：引入空间层次注意（SHAB）与可变形特征交互（DA-AIFI）。
- **Neck**：设计频率-空间特征金字塔网络（FSFPN），核心单元为跨域频率-空间块（CFSB）。

### 2. 关键模块及思想

#### (1) Spatial Hierarchical Attention Block (SHAB)
- 目的：缓解下采样造成的空间特征侵蚀。
- 技术要点：
  - 在改进的 C2f 结构中嵌入轻量自注意机制（SHSA）。
  - 通道对称分割策略（r = 0.5），仅对部分通道施加注意力，降低计算复杂度。
  - 在保留梯度流的同时增强全局依赖。
- 效果：强化小目标在深层特征中的语义保持。

#### (2) Deformable Attention-based Intra-scale Feature Interaction (DA-AIFI)
- 目的：减少 dense scenes 中的噪声干扰与几何错位。
- 技术思路：
  - 利用可变形注意力动态采样邻域 K 点。
  - 计算复杂度从 O(N²) 降为 O(NK)，显著提升效率。
  - 改善空间对齐与定位精度。

#### (3) Frequency-Spatial Feature Pyramid Network (FSFPN)
- 目的：同时强化低层细节与高层语义。
- 核心模块：**Cross-domain Frequency-Spatial Block (CFSB)**  
  由两个分支组成：
  - **空间分支**：使用 Scharr 算子提取边缘方向梯度，经卷积残差融合得到空间增强特征。  
    公式（简要表达）：
    > X_spatial = Conv(Conv(Gx(X)+Gy(X)) + X)
  - **频率分支**：利用 2D DFT + 可学习卷积滤波器 M 抑制背景，强化高频纹理；再经逆 DFT 重建。  
    公式：
    > X_freq = Conv(F⁻¹(Conv(F(X))))
  - **融合机制**：
    > X_out = Conv(X_spatial + X_freq)
- 效果：平衡细粒度边缘与全局纹理信息，提升小目标可辨性。

### 3. 损失函数优化

总损失：
> L = λ_cls·LVFL + λ_L1·LL1 + λ_IoU·LFocaler-EIoU  
其中：
- 分类：使用 **Varifocal Loss (VFL)**，突出高质量样本；
- 回归：结合 **L1** 与 **Focaler-EIoU**，强化低重叠样本的梯度重构；
- 提升了小目标定位的鲁棒性。
