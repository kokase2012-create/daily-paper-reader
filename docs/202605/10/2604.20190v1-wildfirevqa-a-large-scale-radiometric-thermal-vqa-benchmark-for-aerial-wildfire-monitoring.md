## 摘要
野火监测需要来自空中平台的及时且可操作的态势感知，但现有的空中视觉问答（VQA）基准并未评估基于热测量的野火特定多模态推理。我们提出了 WildFireVQA——一个面向空中野火监测的大规模 VQA 基准，集成了 RGB 图像与辐射测温数据。WildFireVQA 包含 6,097 个 RGB-热数据样本，每个样本包括一幅 RGB 图像、一幅彩色映射的热可视化图以及一个辐射热 TIFF 文件，并配有 34 个问题，总计 207,298 个多项选择题，涵盖存在与检测、分类、分布与分割、定位与方向、跨模态推理，以及面向作业型野火情报的飞行规划等多个任务类别。为提高标注可靠性，我们结合了基于多模态大语言模型（MLLM）的答案生成、基于传感器的确定性标注、人工核验，以及帧内与帧间一致性校验。我们进一步建立了一个全面的评估协议，利用辐射热统计，在 RGB、Thermal 及检索增强设置下，对代表性 MLLMs 进行评测。实验结果表明，在各个任务类别中，RGB 仍是当前模型最强的模态，而检索到的热上下文可为更强的 MLLMs 带来性能提升，这既凸显了基于温度的推理的重要性，也揭示了现有 MLLMs 在安全关键性野火场景中的局限性。数据集和基准代码已在 https://github.com/mobiiin/WildFire_VQA 开源。

## Abstract
Wildfire monitoring requires timely, actionable situational awareness from airborne platforms, yet existing aerial visual question answering (VQA) benchmarks do not evaluate wildfire-specific multimodal reasoning grounded in thermal measurements. We introduce WildFireVQA, a large-scale VQA benchmark for aerial wildfire monitoring that integrates RGB imagery with radiometric thermal data. WildFireVQA contains 6,097 RGB-thermal samples, where each sample includes an RGB image, a color-mapped thermal visualization, and a radiometric thermal TIFF, and is paired with 34 questions, yielding a total of 207,298 multiple-choice questions spanning presence and detection, classification, distribution and segmentation, localization and direction, cross-modal reasoning, and flight planning for operational wildfire intelligence. To improve annotation reliability, we combine multimodal large language model (MLLM)-based answer generation with sensor-driven deterministic labeling, manual verification, and intra-frame and inter-frame consistency checks. We further establish a comprehensive evaluation protocol for representative MLLMs under RGB, Thermal, and retrieval-augmented settings using radiometric thermal statistics. Experiments show that across task categories, RGB remains the strongest modality for current models, while retrieved thermal context yields gains for stronger MLLMs, highlighting both the value of temperature-grounded reasoning and the limitations of existing MLLMs in safety-critical wildfire scenarios. The dataset and benchmark code are open-source at https://github.com/mobiiin/WildFire_VQA.

## 二、方法论与技术框架

### 核心思想
- 构建一个融合 **视觉（RGB）与物理温度感知（Thermal Radiometric）** 的大规模多模态 VQA 数据集。
- 结合 **多模态大语言模型（MLLM）自动生成答案** 与 **传感器驱动的确定性标签**, 实现高可靠度标注。

### 数据与结构
- 总计 **6,097** 个配对样本（RGB + 热成像），每个样本包含：
  - 一张 RGB 图像
  - 一张伪彩热图
  - 一张 **辐射热 TIFF 文件（含像素温度值）**
- 每个样本配有 **34 个多项选择问题** → 共计 **207,298 个 QA 实例**。

### 问题类别
共六类任务，涵盖从检测到决策：
1. 存在与检测（Presence & Detection）  
2. 分类（Classification）  
3. 分布与分割（Distribution & Segmentation）  
4. 定位与方向（Localization & Direction）  
5. 跨模态推理（Cross-modal Reasoning）  
6. 飞行规划（Flight Planning）

### 关键算法流程（文字描述）
- **热热点检测算法**
  1. 每像素温度阈值定义：若 `T(x, y) ≥ 200°C` 视为热活跃像素；
  2. 利用 **8 邻域连通分量分析** 获取热点区域；
  3. 结合 UAV 高度与摄像头视场计算地面尺度；
  4. 计算每个热点面积并定义最小有效半径（r_min = 0.75m）；
  5. 基于空间聚类（单链接）识别孤立热源（与主火区距离 > 30m）。

- **空间分布与形态**
  - 通过 **PCA 分析热点质心** 判断线性（火线）结构（线性度阈值 τ_lin = 0.9）。
  - 定义分布标签为 *Linear / Concentrated / Scattered / No-hotspots*。

- **强度一致性计算**
  - 采用最大温度 `I_i = max(T(x,y))` 表征热点强度；
  - 使用中位数绝对离差 (MAD) 计算变异系数，分类为 *相似* 或 *差异显著*。

- **确定性标注模块**
  - UAV 高度由 EXIF 中 GPS 数据与 SRTM 地形数据计算；
  - 紧急车辆检测由 **YOLO-World** 的开放词汇目标检测模型自动实现。

- **多阶段质量控制**
  - *帧内逻辑一致性*（如“无火”场景不应出现“热点位置”标签冲突）。
  - *帧间一致性*：使用 **ORB 特征匹配 + RANSAC** 检测视觉重复帧并保持标签一致。

## 三、实验设计

### 数据与场景
- 主要建立在 **FLAME 3 数据集**：包含三个区域的无人机火烧实验（Sycan Marsh, Willamette, Shoetank），提供同步的 RGB 与辐射热数据。

### Benchmark 设置
- 输入模态：  
  1. RGB  
  2. Thermal  
  3. RGB + Thermal  
  4. 加入 **Radiometric Retrieval Augmentation (RAG)**：附加温度统计（如最大值、标准差、超过 200°C/400°C 像素比例）作为文本上下文。

### 模型对比
- 使用四个代表性多模态大语言模型（MLLM）：
  1. **Qwen3-VL-8B-Instruct**
  2. **LLaVA-v1.6-Mistral-7B**
  3. **InternVL2-8B**
  4. **MiniCPM-V2**

### 实验任务
- 针对六大类任务进行零样本（zero-shot）评估；
- 使用**准确率（Accuracy）**作为主要指标；
- 同时报告随机猜测基线与各类别困难度。

## 六、主要结论与发现

- RGB 仍为当前 MLLM 的主要优势模态；  
- **Qwen3-VL-8B** 在 RAG-RGB 模式下表现最佳（总体准确率 54.76%）；  
- 检索增强（RAG）对强模型有效提升性能（尤其在 Thermal 模式中提升 5%+）；  
- 通过温度统计显式提供物理上下文，可显著改善与温度相关问题的推理（如热点温度、主火行为识别）；  
- MLLMs 在视觉语义解释上表现尚可，但在**物理量感知与定量推理**方面仍受限。
