## 摘要
海洋机器人进行海洋垃圾检测对于生态保护至关重要，但由于图像质量低下（包括模糊、复杂背景和小目标），检测性能往往受限。为应对这些挑战，我们提出了一种改进的基于 YOLO 的检测框架 YOLO-MD。该框架包含一个双分支卷积增强自注意力（Dual-Branch Convolutional Enhanced Self-Attention，DB-CASA）模块，用于强化空间—通道交互，从而提升退化图像中的特征表示。此外，我们引入了一种轻量级基于位移的操作，以在保持参数高效的同时，增强不同尺度目标的细粒度特征提取。进一步地，我们提出了 SFG-Loss，通过动态样本重加权来缓解类别不平衡与优化不稳定问题。在 UODM 数据集上的实验结果表明，YOLO-MD 实现了 0.875 的精度、0.822 的 F1 分数和 0.849 的 mAP50，优于最新的先进方法。该方法的有效性还通过实际的机器人边缘部署实验得到验证。

## Abstract
Marine debris detection for ocean robot is crucial for ecological protection, yet performance is often degraded by low-quality images with blur, complex backgrounds, and small targets. To address these challenges, we propose YOLO-MD, an enhanced YOLO-based detection framework. A Dual-Branch Convolutional Enhanced Self-Attention (DB-CASA) module is designed to strengthen spatial-channel interactions, improving feature representation in degraded images. Additionally, a lightweight shift-based operation is introduced to enhance fine-grained feature extraction for objects of varying scales while maintaining parameter efficiency. We further propose SFG-Loss to mitigate class imbalance and optimization instability via dynamic sample reweighting. Experiments on the UODM dataset demonstrate that YOLO-MD achieves 0.875 precision, 0.822 F1-score, and 0.849 mAP50, outperforming the latest state-of-the-art methods. The effectiveness of this method has also been verified through real-world robotic edge deployment experiments.

## 二、方法论：YOLO-MD 框架

### 2.1 核心思想
在 YOLO 系列架构基础上，论文提出 **YOLO-MD**（Marine Debris Detector），通过三大创新模块提升退化图像下的小目标检测能力与类别均衡性。

### 2.2 关键技术结构

1. **DB-CASA 模块（Dual-Branch Convolutional Enhanced Self-Attention）**
   - 目标：强化空间与通道双向特征交互，在模糊图像中保持细粒度信息。
   - 机制：
     - **双分支结构**：将输入特征分为空间分支和通道分支。
     - 空间分支利用深度可分卷积（DWConv）+ReLU+BN 提升边界与纹理细节。
     - 通道分支采用卷积与全局平均池化（GAP）提取跨通道依赖。
     - 最后通过 **加性融合（Additive Fusion）** 综合空间注意与通道注意，替代高开销矩阵操作，实现高效全局依赖建模。

2. **FSFM 模块（Feature Shift Fusion Module）**
   - 目标：通过无参数的位移（Shift）操作强化局部特征交互。
   - 机制：
     - 将输入特征沿通道分组后，分别进行四向（水平、垂直）位置移动。
     - 移动后的特征再通过通道级拼接（Concat）融合，获得多方向局部上下文。
     - 集成于 C3K2 结构中，并结合残差连接，保持梯度传播畅通。
   - 优势：无需额外卷积参数即可增强局部特征建模能力。

3. **SFG-Loss（Slide-Focal Generalized IoU Loss）**
   - 目标：缓解类不平衡与训练不稳定问题。
   - 机制：
     - 以 **Generalized IoU** 为基损失基础。
     - 引入 **动态样本加权** 函数：对困难样本赋予更高权重，削弱易样本影响。
     - 结合 **Focaler-IoU**机制，限制过度偏重或过低权重样本，防止梯度震荡。
   - 效果：提升收敛稳定性与小目标检测精度。
